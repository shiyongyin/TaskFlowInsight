## 1. 模块定位

`tfi-examples` 是 TFI 产品的 **第一接触点**，直接影响用户对 TFI 的第一印象和采纳意愿。

| 职责 | 重要度 | 完成度 |
|------|:------:|:------:|
| 快速入门引导 | ⭐⭐⭐⭐⭐ | 70% |
| API 用法演示 | ⭐⭐⭐⭐⭐ | 60% |
| 最佳实践传递 | ⭐⭐⭐⭐ | **40%** |
| 性能基准参考 | ⭐⭐⭐ | 80% |
| 集成样板项目 | ⭐⭐⭐ | 50% |

**产品风险**: 作为用户首次接触的代码，tfi-examples 中 **56% 的代码使用内部 API** 而非官方 Facade，严重违背"零侵入"产品定位，可能让新用户误入歧途。

## 2. 目标用户

### 按技术水平

| 用户 | 关注点 | 对应内容 | 体验 |
|------|--------|----------|:----:|
| **新手开发** | 5 分钟跑通 | Chapter 1-2 + Demo01-04 | ⭐⭐⭐⭐ |
| **中级开发** | 变更追踪 + 比对 | Chapter 5-6 + Demo05-07 | ⭐⭐ |

**问题**: 中级用户的核心学习路径（Demo05-07, CT Demo）体验最差。

### 按使用场景

| 场景 | 用户操作 | 当前支持 |
|------|----------|:--------:|
| CLI 交互演示 | `./mvnw exec:java` | ✅ |
| Spring Boot REST | `./mvnw spring-boot:run` + curl | ✅ |
| 复制代码到自己项目 | 参考 Demo 源码 | ⚠️ 内部 API 误导 |
| 性能评估 | JMH 基准 | ✅ |
| 学习 @TfiTask 注解 | 查看 AnnotationDemo | ⚠️ 无 Javadoc |

快速   电商   高级   最佳   高级   变更   异步

## 5. Compare Demo 用户体验

### 5.1 渐进难度分析

| Demo | 难度 | 行数 | 用户友好度 | 原因 |
|------|:----:|:----:|:----------:|------|
| Demo01 | ★☆☆ | 225 | ⭐⭐⭐⭐ | Facade API, 简单类型 |
| Demo02 | ★☆☆ | 274 | ⭐⭐⭐⭐ | Facade API, 日期 |
| Demo03 | ★★☆ | 255 | ⭐⭐⭐⭐ | Facade API, @Entity |
| Demo04 | ★★☆ | 164 | ⭐⭐⭐⭐ | Facade API, 集合 |
| Demo05 | ★★★ | 249 | ⭐⭐⭐⭐ | Facade API, Entity List |
| **Demo05_List** | ★★★★★ | 641 | ⭐⭐ | **突然底层 API, 641 行** |
| **Demo06** | ★★★★★ | 813 | ⭐⭐ | **813 行, 反射代码** |
| **Demo07** | ★★★★★ | 779 | ⭐⭐ | **779 行, 模型重复** |

**学习曲线断崖**: Demo05 (249行, Facade) → Demo05_List (641行, 底层API) 难度跳跃过大。

### 5.2 Demo 命名困惑

| 问题 | 说明 |
|------|------|
| Demo05 vs Demo05_List | 两个都是集合实体比对，命名不清晰 |
| Demo01 vs Demo01_org | _org 后缀含义不明，实为旧版本 |
| 无统一前缀 | 有些用 `Demo0x_`，有些没有 |

## 6. 双入口体验

### 6.1 CLI 入口

| 优势 | 问题 |
|------|------|
| 交互式菜单 | 菜单仅 6 项（遗漏第 7 章） |
| 渐进式演示 | 反射初始化 TfiCore |
| 可指定章节运行 | Demo01-07 无法从菜单直达 |

### 6.2 Spring Boot 入口

| 优势 | 问题 |
|------|------|
| REST 端点可 curl | 仅 4 个端点，覆盖不全 |
| Actuator 可用 | 无 Swagger/OpenAPI 文档 |
| AOP 注解生效 | DemoController 无输入校验 |

### 6.3 用户触达矩阵

| 功能 | CLI 可达 | REST 可达 | 独立运行可达 |
|------|:--------:|:---------:|:------------:|
| Ch1-6 | ✅ | ❌ | ❌ |
| Ch7 | **❌ 未注册** | ❌ | ❌ |
| Demo01-07 | ❌ | ❌ | ✅ `mvn exec:java` |
| REST 端点 | ❌ | ✅ | ❌ |
| JMH 基准 | ❌ | ❌ | ✅ `-P bench` |

**问题**: 功能分散在 3 种入口，用户不知道从哪里开始。
